ATM_Simulation

A simple terminal-based ATM system built using Python. This project demonstrates basic banking operations using modular programming, lists, and dictionaries.

🚀 Features

- Check Account Balance  
- Deposit Money  
- Withdraw Money  
- View Transaction Statement  
- Menu-driven interface  

🛠️ Tech Used

- Python  
- List & Dictionary (for data storage)  
- Modular Programming (multiple files)

📁 Project Structure

ATM_Simulation/
│
├── main.py
├── deposit.py
├── withdraw.py
├── check_balance.py
├── show_statement.py
├── utils.py

▶️ How to Run

1. Clone the repository  
2. Open terminal in project folder  
3. Run:

python main.py

🧠 How It Works

- Account data is stored using a dictionary  
- Transactions are stored in a list  
- Each functionality is separated into different modules  
- The program runs using a loop-based menu system
